List of GIT commands:

GIT command Description
git init initializes directory into a Git repo
git add adds files in the staging area for Git
git commit -m "input message here" records the changes made to the local repo
git status returns the current working branch
git config used to assign config settings
git branch branch_name returns the current branch the local repo is on
git branch -a lists all remote or local branches
git branch -d branch_name deletes a branch
git checkout branch_name switches to another branch
git checkout -b branch_name creates and switches to new branch
git merge merges changes into current branch
git clone remote_url creates local working copy of an existing remote repo
git pull gets latest version from remote repo to local repo
git push pushes local commits to remote repo
git stash -u stores current work with untracked files
git stash pop brings stashed work back to the working directory
git log shows entire git log and commit id (SHA)
