HTML Notes:

Content Models:

  1. Metadata content: modify the presentation or the behavior of the rest of the document, set up links to other documents, or convey other out of band information.
  2. Flow content:contain text or embedded content
  3. Sectioning Content: create a section in the current outline that defines the scope of header/footer elements and heading content
  4. Heading Content: Defines the title of a section
  5. Phrasing Content: Content defines the text and the markup it contains
  6. Embedded Content: imports another resource or inserts content from another markup language or namespace
  7. Interactive Content: specifically designed for user interaction

Semantic Elements:

Note: all of these elements are block level elements

  1. Header contains header information about the page (ie. company logo, tag line,etc)
  2. nav often contained within the header and signifies content used for navigation within website
  3. h1
  4. section defines a section in a document
  5. artcile often inside the section element.
  6. aside relates to the main content of the page
  7. footer contains footer information
